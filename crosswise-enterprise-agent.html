🚀 CrossWise 三大核心创新

1. 关系管理（Relationship）

  • Agent 需要认识团队所有人、了解他们的角色
  • 知道其他 Agent 能做什么
  • 支持自动导入公司组织架构

2. 自主意识（Aware）

  • 批评 OpenClaw 默认心跳 30 分钟太慢
  • CrossWise 让 Agent 自主规划工作节奏，扫描间隔 15 秒
  • 支持触发器组合：定时 + on_message + 条件判断
  • 示例：催发票场景——"每分钟提醒直到回复，回复后取消触发器并通知 admin"

3. 协作机制

  • 多人共用 Agent 场景：收集团建意见、汇总后交给财务
  • 审批控制：Agent 做什么操作需要审批
  • 成本控制：细颗粒度控制每人每天的 token 消耗、Agent 生命周期、调用频率
