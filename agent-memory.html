🧠 Agent Memory 技术笔记

从"失忆"到"进化"的记忆系统演进
三维分类框架 AutoDream Memento-Skills Mem0

Agent 记忆系统不是简单的"存储-检索"，而是形式×功能×动态的三维架构。核心演进路径：从"失忆"→"自愈"→"进化"，关键突破是"睡眠整理"机制和"技能即记忆"范式。

一、三维分类框架

维度分类说明技术实现
形式词元级文本 token 存储RAG、向量数据库
参数记忆内化于模型权重微调、训练
潜在记忆隐藏状态压缩RNN hidden state
功能事实记忆"是什么"的知识世界知识、领域知识
经验记忆"怎么做"的知识技能、策略、教训
工作记忆临时上下文对话历史
动态形成→演化→检索记忆生命周期编码、巩固、遗忘、提取

二、问题诊断

多 Agent 记忆隔离问题

问题根因影响
上下文丢失Agent 实例独立每次重启"失忆"
重复沟通无共享记忆层反复交代背景
Token 消耗高全量记录无筛选成本失控
记忆无重点无差别保存关键信息淹没

记忆隔离是特性不是 bug：保护隐私和专注度，问题在缺少共享层。

三、解决方案对比

AutoDream

后台整理机制，像人类睡眠一样周期性压缩、修剪、刷新记忆

merge → prune → refresh → compact

Memento-Skills

技能即记忆，通过"读-写-反思"循环实现无需重新训练的学习

GAIA +13.7%, HLE +20.8%

Mem0

向量检索 + 关键信息捕捉，支持跨会话/跨 Agent

Token 消耗低，检索效率高

四、演进路径

阶段1: 失忆（无记忆） ↓ 阶段2: 临时记忆（单会话） ↓ 阶段3: 持久记忆（跨会话） ↓ 阶段4: 自愈记忆（AutoDream） ↓ 阶段5: 进化记忆（技能即记忆）

突破说明代表方案
